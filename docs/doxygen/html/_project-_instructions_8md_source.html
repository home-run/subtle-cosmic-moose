1 # Take Me Out the Ballgame
2 
3 >A list of all major league baseball stadiums resides at http://en.wikipedia.org/wiki/List_of_Major_League_Baseball_stadiums. A list of all the major league baseball teams reside at http://mlb.com. Your software team is to write a program that stores this data in a suitable data structure. (map, vector, list, heap, BST, tree, etc.) Your application will allow baseball fans to plan their dream vacation.

13 Requirements:
14 
15 1. Write at least 10 agile stories (including description, tasks, test scenarios, and story points) before any software development.
16 2. You should create a list of the major league stadiums by name in alphabetical order.
17 3. You should create a list of major league stadiums sorted by team name.
18 4. You should create a list of American League stadiums sorted by team name.
19 5. You should create a list of National League stadiums sorted by team name.
20 6. You should create a list of stadiums that have a grass surface sorted by team name.
21 7. You should create a list of stadiums in chronological order by date opened
22 8. You should create a list of stadiums sorted by seating capacity.
23 9. You should create a list of stadiums sorted by park typology.
24 
25 Planning a vacation:
26 
27 1. Design trips to visit each major league stadium travelling the shortest distance starting at Dodger stadium using Dijkstra’s algorithm (note: each trip begins at Dodger stadium).
28 2. Provide the capability for a baseball fan to plan his/her dream vacation by allowing a baseball fan to choose their starting stadium and all the other stadiums they would like to visit. Determine the order the stadiums will be visited. Use Dijkstra’s algorithm to calculate the shortest distance for this dream vacation.
29 3. Determine the minimum spanning tree (MST) connecting all the major league stadiums using Prim’s or Kruskal’s algorithm. Calculate the associated mileage.
30 4. Provide the capability to track souvenirs purchased by the baseball fans on their trips and the corresponding costs.
31 5. Output the total revenue for each stadium including a grand total for all the stadiums.
32 6. Provide the ability to modify stadium information if a team moves into a new stadium. (administrator only) (Atlanta Braves are planning to move to a new stadium in 2017).
33 7. Provide the ability to add a new team and corresponding stadium if major league baseball decides to add new teams. (administrator only)
34 8. You should be able to randomly access any baseball stadium.
35 9. You should be able to modify (add/delete/change) the souvenirs list and their prices. (administrator only)

No late projects will be accepted. Your team must demonstrate your project to me before it will be graded. Each teammate must identify their accomplishments on the project. Not all team members will necessarily earn the same score.
